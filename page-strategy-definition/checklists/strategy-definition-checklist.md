
# Strategy Definition Checklist

## Strategic Integrity

- page mission defined
- user state defined
- conversion goal defined
- narrative rationale included

---

## Narrative Quality

- section order justified
- user psychology considered
- objection timing appropriate
- no random section stacking

---

## Information Hierarchy

- primary messaging prioritized
- secondary information separated
- above-the-fold clarity defined
- cognitive overload reduced

---

## Conversion Strategy

- CTA logic explicit
- trust strategy explicit
- friction level appropriate
- objections addressed

---

## Output Validation

- no UI generation
- no fake certainty
- assumptions preserved
- strategic risks identified
- content drafting readiness declared
