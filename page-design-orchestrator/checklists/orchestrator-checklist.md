
# Orchestrator Checklist

## Workflow Integrity
- current state identified
- active step identified
- invalid transitions prevented
- routing decision explained

## Artifact Integrity
- required artifacts exist
- assumptions preserved
- blockers visible
- accepted risks visible
- decisions traceable
- revisions traceable

## Readiness Integrity
- readiness status explicit
- blockers categorized
- rollback conditions checked
- downstream risks visible

## Alignment Integrity
- multi-role review checked
- unresolved concerns checked
- tradeoffs checked
- ownership checked
- alignment requirement assessed before handoff

## Output Validation
- no skipped steps
- no hidden assumptions
- no invalid routing
- no skipped alignment when material conflicts exist
- workflow continuity preserved
- next-step recommendation included
