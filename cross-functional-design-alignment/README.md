
# cross-functional-design-alignment

This skill facilitates collaborative alignment after multi-role review and before designer handoff.

Primary objective:
convert critique into agreed decisions, accepted tradeoffs, implementation boundaries, and handoff-ready alignment.

Recommended workflow:

1. page-task-intake
2. business-context-structuring
3. page-requirement-gap-analysis
4. page-requirement-discovery-qna
5. page-strategy-definition
6. content-first-page-draft
7. wireframe-specification
8. multi-role-review
9. cross-functional-design-alignment
10. designer-handoff-brief
11. workflow-retrospective

Best used when:
- multiple disciplines need to agree before design execution
- engineering feasibility matters
- stakeholder concerns exist
- design handoff should not inherit unresolved conflict
