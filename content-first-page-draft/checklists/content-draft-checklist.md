
# Content Draft Checklist

## Section Integrity

Every section:
- has explicit purpose
- supports strategy
- maps to user state
- advances conversion logic

---

## Messaging Quality

- hierarchy is clear
- messaging is non-redundant
- narrative continuity exists
- information density appropriate

---

## CTA Logic

- primary CTA consistent
- CTA timing appropriate
- friction level aligned
- escalation strategy coherent

---

## Visual Asset Logic

- visuals support understanding
- assets justified
- unnecessary visuals removed

---

## Output Validation

- no UI generation
- no aesthetic overreach
- no generic filler sections
- assumptions explicit
- wireframe readiness declared
