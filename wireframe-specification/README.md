
# wireframe-specification

This skill converts structured content drafts into low-fidelity wireframe specifications before UI design begins.

Primary objective:
validate layout, hierarchy, and UX structure before visual exploration.

Recommended workflow:

1. page-task-intake
2. business-context-structuring
3. requirement-gap-analysis
4. requirement-discovery-qna
5. page-strategy-definition
6. content-first-page-draft
7. wireframe-specification
8. multi-role-review

Best used for:
- landing pages
- SaaS product pages
- onboarding flows
- pricing pages
- conversion-focused websites
