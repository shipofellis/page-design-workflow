
# Wireframe Checklist

## Structural Integrity

Every section:
- has layout rationale
- preserves hierarchy
- supports narrative flow
- supports conversion logic

---

## Responsive Integrity

- mobile behavior defined
- hierarchy preserved
- stacking rules explicit
- CTA adaptation defined

---

## UX Quality

- scanning flow clear
- density manageable
- CTA visibility sufficient
- trust placement logical
- interaction complexity controlled

---

## Output Validation

- no high-fidelity UI
- no branding systems
- no aesthetic overreach
- assumptions explicit
- UI exploration readiness declared
