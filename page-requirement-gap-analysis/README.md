
# page-requirement-gap-analysis

This skill audits page-design requirements before page strategy work begins.

Primary objective:
prevent weak or incomplete inputs from contaminating downstream strategy and design decisions.

Recommended workflow:

1. page-task-intake
2. business-context-structuring
3. page-requirement-gap-analysis
4. requirement-discovery-qna
5. page-strategy-definition

This skill should be used by:
- product managers
- UX strategists
- AI workflow systems
- design operations pipelines
